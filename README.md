## Getting Started

Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000] with your browser to see the result.

On a separate terminal run the sanity dev server:

```bash
cd sanity
npm run dev
```

Open [http://localhost:3333] with your browser to see the result.

## Architecture

This project demonstrates a simple movie catalog application built with Next.js and a headless CMS.

Tech stack:

- Next.js (App Router)
- Typescript
- React
- Sanity CMS
- Tanstack React Query
- Tailwind / SCSS

The catalog UI fetches product data from Sanity using server-side queries.
Content can be managed through the Sanity Studio.
